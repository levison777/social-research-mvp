研究问题

用户如何讨论 AIGC 工具在内容生产中的价值、风险和使用门槛？

数据范围

样本包含 248 条内容记录、1864 条评论或回复、58 个高频账号，路由覆盖 11 个平台和兜底网页。

关键发现

  • 讨论热点集中在效率、真实性、版权、价格四类主题。
  • 评论区与正文强相关比例为 63%，跑题或玩梗约 13%。
  • 平台语境差异明显，适合输出分平台结论。

局限

当前 MVP 以 opencli 与已配置 API 平台采集为主，Google News 会受网络可达性影响，LinkedIn Link 评论优先走 Apify actor，未配置时再依赖浏览器登录态和页面可见评论；全量 hashtag 统计仍建议接官方 API 或合规数据供应商。
